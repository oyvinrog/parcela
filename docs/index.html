How It Works

Four simple steps to unbreakable security

1

Encrypt

Your file is encrypted with AES-256-GCM using your password

2

Split

The encrypted blob is split into 3 shares (any 2 can reconstruct)

3

Distribute

Store shares in different locations ‚Äî USB, cloud, safe

4

Recover

Combine any 2 shares + your password to decrypt
